How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

This site is the Stadium High School version of 2048. Mr. Elstein created it by forking the original source code on GitHub and then modifying the CSS. To learn how to create your own version of 2048, take this short, free course on Udacity. If you want to learn more about creating cool web applications, sign up for Exploring Computer Science or AP Computer Science A for the next school year.

The original 2048 was created by Gabriele Cirulli, based on 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer.

If you think you've found a bug in the application or have ideas for how to make it better, please contact Mr. Elstein (jelstei@tacoma.k12.wa.us).
